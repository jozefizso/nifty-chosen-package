# Nifty Template - Chosen plugin

> Chosen is a library for making long, unwieldy select boxes more user friendly.

Chosen plugin style from the Nifty Admin Dashboard template.

### Bower Installation

Chosen is available for Bower installation:
`bower install nifty-chosen`
